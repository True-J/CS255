# CS255
The DriverPass project was a project for the CS-255 class at SNHU.

The project was an examaple of a client who had a business that helped teach people how to pass tests from the DMV. The client wanted to have a website that would allow students to take practice tests, schedule driving instructions, and pay for various packages. The client wanted to be able to create pricing packages, and connect to the DMV through an API to create practice tests and driving rules updates.

I think my activity diagrams were the best work I did on this project. I think I did a good job breaking down each step into anclear structure that could be easily programed. I think my sequence diagrams and my gannt charts could have used some work to improve. I think the sequence diagram could have more clear separation for the loops. For the gannt chart, I think I should have used a vertical alignment for the subjects rather than using a horizontal one based on the teams.

The user was pretty clear for what they wanted. The website is a very simple display for information in a database. The main set up was to sepeeate each user into a class that had different read and update permissions to the database. The database would need to be updated regularly with DMV tests and rules. 

In an everyday situation, a user may not fully understand how difficult implementation for their ideas may be, and they may not know how certain elements will need to interact with each other. For example, the DriverPass CEO may not understand that a User is not just his clients, but that we can use class inheritance so that the Clients, CEO, Secretary, and IT Soecialist all have different permissions. So it is necessary to understand not just the elements that they want in their product but also elements that they may need and not know they need. Like having some form of login verification.

I approach designing software by outlining what the end output should look like first, and working backwards to the input. That way I know what my eventual goal is for each section of the code. This class did really help me to find ways to organize my thoughts better for my personal projects as well as in my career. I plan in getting better with the various diagrams we used in this class so I can better utilize them when starting projects.
